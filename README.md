## ✨ ใหม่ในเวอร์ชันนี้

1. **หน้าค้นหา = 2 แท็บ**
   - 🔵 **แมวที่คนเจอ** (เปิดมาเจอแท็บนี้ก่อน) — เจ้าของที่ตามหา มาดูว่าแมวตัวเองถูกใครเจอไหม
   - 🔴 **แมวหาย** — โพสต์ของเจ้าของที่ประกาศตามหา
   - กดโพสต์เข้าไปดูรายละเอียดเต็มได้ทั้ง 2 แท็บ
2. **Match AI คิด % เฉพาะแมวหายเท่านั้น** (ไม่เอาแมวอยู่บ้านมาปน)
3. **เจอแมวจร → เลือก 2 แบบ**
   - 🔴 มั่นใจว่าแมวหาย (มีปลอกคอ/พันธุ์เลี้ยง/มาอาศัยตามบ้าน)
   - 🟡 อาจเป็นแมวจร หรือแมวหาย (ไม่แน่ใจ)
4. **แผนที่ซูมเข้า = เห็นรูปแมว** (รูปที่คนถ่ายตอนเจอ)
   - ซูมออก = จุดสี / ซูมเข้า = รูปแมวกลม ๆ มีกรอบสี
   - 🔴 แดง = แมวหาย/มั่นใจว่าหาย, 🟡 เหลือง = อาจเป็นจร/หาย
5. ทุกโพสต์ **เรียงตามระยะใกล้ตัวเรา**

## 📦 ลง / รัน

แตก zip วางทับโฟลเดอร์ กด Replace ทั้งหมด (firebase.js ไม่โดนแตะ)

ไลบรารี: **ใช้ของเดิม ไม่ต้องลงเพิ่ม** (ถ้ายังไม่เคยลง: `npx expo install react-native-maps expo-location expo-linear-gradient`)

```bash
npx expo start -c
```

> 🔥 Firestore ต้องเป็น test mode (เปิด) — มีคอลเลกชัน `sightings`

## 🎬 เดโม่

1. ลงทะเบียนแมวสีต่างกัน 4-5 ตัว
2. แจ้งหาย 2-3 ตัว (กรอกรางวัล + อนุญาต GPS)
3. กด "เจอแมวจร" → ถ่าย/เลือกรูปแมวส้ม → เลือกสีส้ม → เด้ง "🎯 น่าจะเป็นแมวหาย!"
   → เลือก "🔴 มั่นใจว่าแมวหาย" → ปักหมุด
4. กลับหน้าค้นหา แท็บ "🔵 แมวที่คนเจอ" → เห็นโพสต์ที่เพิ่งปัก
5. เปิดแผนที่ → ซูมเข้า → เห็นรูปแมวจริงบนหมุด

## 🗺️ แผนที่ Android

```json
{ "expo": { "android": { "config": { "googleMaps": { "apiKey": "ใส่_KEY" } } } } }
```
